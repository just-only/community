## git
 #### 基本使用
 ###### git add .  添加当前文件夹所有文件到仓库,之后再提交
 ###### git commit -m "描述"
 ###### git remote add origin 远程地址
 ###### git push origin 分支名  -f 强制推送，对于不是基于远程库修改的
 ###### git commit --amend --no-edit 不修改备注的追加，在向远程推送之前
 ###### git status 查看状态
 ###### git pull 当远端的代码和本地不同时,使用命令将远程代码拉到本地，之后再进行提交
